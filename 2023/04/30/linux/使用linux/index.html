各类小技巧

  1. 强制停止

    • Ctrl+c 退出程序

    • Ctrl+c 退出登出或退出

    不能用于退出vi/vim

  2. 历史命令查找

    • history

      过滤使用的指令里带有ch的命令

    • Ctrl+r 输入内容匹配

      如果搜索到的命令是可执行的，则直接回车

      键盘的左右键，可以得到此命令

    • ! 匹配最近的

      比如：

      匹配最近的p开头的命令

  3. 光标移动

    • Ctrl + a 跳到命令开头
    • Ctrl + e 跳到命令结尾
    • Ctrl + 左键 向左跳过一个单词
    • Ctrl + 右键 向右跳过一个单词

  4. 清屏

    • Ctrl + r/clean
